conf.sh

Freight configuration. Default directories for the Freight library and Freight cache. Your web server's document root should be $VARCACHE.

Default Origin and Label fields for Release files.


VARLIB="/var/lib/freight"
VARCACHE="/var/cache/freight"

Parse the configuration file for overrides. Typically, the config file would be found at $prefix/etc/freight.conf but a special exception is made when $prefix is /usr. In that case, the config file is /etc/freight.conf.

ORIGIN="Freight"
LABEL="Freight"

Normalize directory names.

[ -z "$CONF" ] && {
	CONF="$(dirname $(dirname $0))/etc/freight.conf"
	[ "$CONF" = "/usr/etc/freight.conf" ] && CONF="/etc/freight.conf"
}
. "$CONF"


VARLIB=${VARLIB%%/}
VARCACHE=${VARCACHE%%/}
