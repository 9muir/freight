Freight

A modern take on the Debian archive

Installation

Clone and make install or install from DevStructure’s Debian archive:

sudo tee /etc/apt/sources.list.d/devstructure.list <<EOF
deb http://packages.devstructure.com/ maverick main
EOF
wget -O - http://packages.devstructure.com/keyring.gpg | sudo apt-key add -
sudo apt-get update
sudo apt-get -y install freight
