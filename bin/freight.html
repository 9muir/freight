freight

freight dispatches to freight-* commands.

Just like the DevStructure tools, which try to be just like Git, accept a subcommand that really just completes the name of a real command.

#!/bin/sh


set -e

Be helpful in this case since no subcommand was found.

COMMAND=$0-$1
[ -x $COMMAND ] && {
	shift
	exec $COMMAND "$@"
}


echo "Usage: $(basename $0) <command> [...]" >&2
echo "Common commands: add cache" >&2
echo "See all available commands by typing \"$(basename $0)-<TAB><TAB>\"" >&2
exit 1
